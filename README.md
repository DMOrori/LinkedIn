# LinkedIn Clone Android App

A professional networking app clone inspired by LinkedIn, built with Kotlin and modern Android development practices.

## Features

- **Authentication**: Fully functional Login and Registration screens with validation.
- **Home Feed**: A scrollable feed of posts with user profiles, content, and interaction buttons (Like, Comment, Repost, Send).
- **Network Management**: View and manage professional connections and suggestions.
- **Bottom Navigation**: Seamlessly switch between Home, My Network, Post, Notifications, and Jobs.
- **Modern UI**: Styled with LinkedIn's signature blue and gray color palette, featuring Material Design components.

## Tech Stack

- **Language**: Kotlin
- **Architecture**: MVVM (Model-View-ViewModel) pattern principles.
- **UI Components**: 
    - `RecyclerView` for efficient list rendering.
    - `BottomNavigationView` for app-wide navigation.
    - `ConstraintLayout` for responsive and complex UI designs.
    - `CardView` and `ShapeableImageView` for modern styling.
    - `TextInputLayout` for enhanced user input experience.
- **Jetpack Libraries**: 
    - `Fragment` for modular UI components.
    - `ViewCompat` for edge-to-edge display support.

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync the Gradle project.
4. Run the app on an emulator or a physical Android device.
